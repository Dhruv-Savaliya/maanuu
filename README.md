# 💖 Maanuuu — A Love Story Website

A deeply personal, cinematic love-story website built with **Next.js, Tailwind CSS V4, and Framer Motion**. Optimized for deployment and smooth, premium animations.

## 🚀 How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎵 Fixing/Adding the Song

The current default song is a beautiful cinematic piano track located at `public/music/song.mp3`.

If you want to use **your own specific song** (e.g., *Tera Ban Jaunga*, *Tum Hi Ho*, etc.):
1. Get the `.mp3` file of the song you want.
2. Rename it to exactly `song.mp3`.
3. Replace the existing file at `public/music/song.mp3` with your new file.
4. Refresh the page and tap the Music Orb to play!

## 🌍 How to Deploy (Vercel)

This Next.js app is perfectly set up for Vercel deployment:
1. Push this entire folder to a **GitHub repository**.
2. Go to [Vercel](https://vercel.com/) and log in.
3. Click **Add New Project** and select your GitHub repository.
4. Leave all settings as default and click **Deploy**.
5. Within 2 minutes, your website will be live and you can share the link with Maanuuu!

## ✏️ Customizing Text
You can edit the text directly in `src/app/page.tsx`. Everything is nicely structured section by section.
